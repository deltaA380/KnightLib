More information coming soon

## Usage
First, add JitPack as a Maven repository in your build.gradle:

```
repositories {
    maven { url 'https://jitpack.io' }
}
```

Then add the dependency:

```
dependencies {
    compile 'com.team2052:KnightLib:1.0.0'
}
```
